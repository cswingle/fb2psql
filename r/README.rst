fb2psql R
=========

The activities aren't easily downloaded using the Python fitbit library,
so I'm getting this data using R.

Requirements
------------

The following R libraries (and dependencies) are required:

* fitbitScraper
* dplyr
* RPostgreSQL
* lubridate
